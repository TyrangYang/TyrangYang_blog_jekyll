## Classification

* Sequence container
    - array (build in)
    - vector
    - heap
    - priority queue
    - list
    - slist (not standard)
    - deque 
    - stack (adopter)
    - queue (adopter)
* Associative container 
    - RB-tree (not public)
    - set
    - map
    - multiset
    - multemap
    - hashtable (not standard)
    - hash_set (not standard)
    - hash_map (not standard)
    - hash_multimap (not standard)
    - hash_multiset (not standard)
